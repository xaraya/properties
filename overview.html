IconDropdown Property

Description

The IconDropdown dataproperty is a dropdown that in its output display shows an image icon. The icons for for the dropdown values can be configured.

Usage

Standalone

    <xar:data-input type="icondropdown" [attr1="value1",...] />
    <xar:data-output type="icondropdown" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['icondropdown']" [attr1="value1",...] />
    <xar:data-output property="$properties['icondropdown']" [attr1="value1",...] />

Configuration Values

  • icon_directory: string [default="set1"] The directory below the property's template directory where the icon files are found.
  • icon_options: string [default="0,red-12.png;1,yellow-12-png;2,orange-12.png;3,green-12.png;4,blue-12.png;5,clear-12.png;"] A simple array description that assigns dropdown values to icon files.
