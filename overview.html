Listing Property

Description

The Listing dataproperty is a standalone widget that lets you display dataobject items in tabular form. It includes options for:

  • Full text search on any of the displayed fields
  • Filtering by initial letter of the key field
  • A pager for listings with many items
The property uses a set of templates that can be selectively overridden to change parts of the display. These include links to add, display, modify and delete. The links by default call the respective GUI functions of the module containing the listing property tag. Clicking on a column header sorts by that column alternately ascending and descending.

The Listing dataproperty only supports objects with flat table datasources. For objects containing dataproperties with other datasources, those properties are ignored.

Usage

Standalone

    <xar:data-input type="listing" [attr1="value1",...] />

Examples

    <xar:data-input type="listing" object="$users" />
    <xar:data-input type="listing" objectname="roles_users" fieldlist="name,uname" />

Attributes

  • object: dataobject A list object the listing is based on
  • objectname: string The name of a list object the listing is based on. This attribute overrides the object attribute (if both are present).
  • fieldlist: string list A comma separated list of property names to be displayed. If this attribute is absent, then all the active fields of the object are displayed. Unrecognized property names are ignored.
  • show_primary: boolean [default=false] Display the primary field of the object in the listing.
  • show_search: boolean [default=true] Display a field for full text search.
  • show_alphabet: boolean [default=true] Display a filter for display by first letter of the key field.
  • showall_tab: boolean [default=true] Display an "All" link on the alpha filter.
  • showother_tab: boolean [default=true] Display an "Other" link on the alpha filter.
  • show_items_per_page: boolean [default=false] Display a dropdown in the pager to allow the user to set the number of items displayed.
